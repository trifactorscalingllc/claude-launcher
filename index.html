AI project summaries

Generate a one-line "what this is + where it stands" per project with Claude (claude-opus-4-8). Summaries are cached and only regenerate when the project changes. Your key is encrypted with your OS keychain — never stored in plaintext.

Real billed usage optional

Card and overview costs are estimates. If your org has an Admin API key (sk-ant-admin…), add it to pull your actual billed month-to-date cost from Anthropic's Cost API into the Overview. Encrypted with your OS keychain. (Pro/Max subscriptions have no per-user billing API — leave this blank.)

Budget & notifications

Get warned in-app and via desktop notification when spend approaches your budget. Set 0 to turn a budget off.

About & updates

Claude Helm runs 100% locally. It reads your Claude Code history on this machine to build the dashboard — no account, no telemetry, nothing leaves your computer. The only optional network call is AI summaries, if you add your own API key above.
